This is my source code, written in C++, from Jamis Buck's book: [The Ray Tracer Challenge](https://pragprog.com/titles/jbtracer/the-ray-tracer-challenge/).


# Usage

```console
$ mkdir ./build
$ cd build
$ cmake ..
$ cmake --build .

# run unit tests
$ ./unit_tests --success

# run raytracer
$ ./raytracer
```
